# Bella Vista Restaurant - Fine Dining Experience

A beautifully animated, fully responsive restaurant website featuring modern design, extensive CSS animations, and seamless user experience. Built with pure HTML5, CSS3, and vanilla JavaScript for optimal performance and easy deployment.

## 🌟 Features

### Design & User Experience
- **Fully Responsive Design** - Works perfectly on desktop, tablet, and mobile devices
- **Modern Restaurant Theme** - Warm, inviting color palette with elegant typography
- **Accessibility Focused** - WCAG guidelines compliant with reduced motion support
- **SEO Optimized** - Semantic HTML structure and meta tags

### Advanced Animations
- **CSS Keyframe Animations** - Smooth 60fps animations using GPU acceleration
- **Scroll-Triggered Effects** - Intersection Observer API for performance
- **Interactive Hover States** - Enhanced user feedback on all interactive elements
- **Loading Screen** - Animated restaurant logo with progress indication
- **Parallax Effects** - Multi-layer background animations
- **Morphing Navigation** - Creative hamburger menu transformations

### Sections & Functionality
- **Hero Section** - Animated title reveals with floating elements
- **About Section** - Animated statistics counters and feature highlights
- **Menu Section** - Filterable menu items with animated category switching
- **Gallery** - Interactive lightbox with keyboard navigation
- **Contact Form** - Real-time validation with animated feedback
- **Interactive Map** - Click-to-view location functionality

### Technical Features
- **Pure CSS Animations** - No external animation libraries required
- **Vanilla JavaScript** - Clean, dependency-free code
- **CSS Custom Properties** - Consistent theming and animation timing
- **Performance Optimized** - GPU-accelerated transforms and opacity changes
- **Progressive Enhancement** - Graceful degradation for older browsers
